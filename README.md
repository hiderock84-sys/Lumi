## 🌟 プロジェクト概要

株式会社ルミライズは、依存症者・刑務所出所者等、行き場のない方々のための住宅提供サービスを行う専門企業です。このウェブサイトは、**Housing First原則**と**エンパシードリブンデザイン**を基に、世界中の支援組織のベストプラクティスを研究して構築されました。

## 🎯 Housing First原則の反映

このウェブサイトは、世界的に認められている**Housing First**アプローチを体現しています：

1. **無条件の住宅提供** - 前提条件なしの受け入れを強調
2. **透明性** - 入居プロセス、費用、条件の明確な開示
3. **包括的サポート** - FAQ、緊急対応、長期フォローアップ
4. **プライバシー保護** - 個人情報保護の明示
5. **エンパシードリブン** - 利用者の立場に立ったコンテンツ

## 🚀 ローカル開発

### 起動方法

#### Python（推奨）
```bash
cd /home/user/webapp
python3 -m http.server 8000
```

#### Node.js
```bash
cd /home/user/webapp
npx http-server -p 8000
```

#### VS Code Live Server
1. VS Codeで`index.html`を開く
2. 右クリック → "Open with Live Server"

ブラウザで `http://localhost:8000` にアクセス
